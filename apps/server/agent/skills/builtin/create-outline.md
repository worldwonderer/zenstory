# 创建大纲

帮助用户创建故事大纲，包括章节结构和情节规划。

## Triggers
- 创建大纲
- 写大纲
- 故事大纲
- 章节大纲
- 情节规划

## Instructions
你现在进入大纲创建模式。请帮助用户规划故事结构。

### 任务目标
创建清晰的故事大纲，规划完整的情节结构，为后续写作提供明确的方向指引。

### 执行流程

1. **确定故事核心**：
   - 故事类型（言情/悬疑/奇幻/现实等）
   - 核心主题和想要表达的内容
   - 目标读者群体

2. **设计三幕结构**：
   - 第一幕（开端）：建立世界、引入角色、触发事件
   - 第二幕（发展）：冲突升级、角色成长、情节推进
   - 第三幕（高潮与结局）：最终对决、解决冲突、收束故事

3. **规划关键节点**：
   - 触发事件：打破日常的事件
   - 第一转折点：主角踏上旅程
   - 中点：重大发现或转变
   - 第二转折点：最黑暗时刻
   - 高潮：最终对决
   - 结局：新的平衡

4. **划分章节**：
   - 根据情节节点划分章节
   - 每章设定明确的目标
   - 章节间保持节奏变化

### 输出格式

使用 `create_file` 工具创建大纲文件，文件类型为 `outline`，格式如下：

```
# [故事标题] 大纲

## 故事概念
- 类型：
- 主题：
- 一句话概括：

## 三幕结构

### 第一幕：开端
（故事的起点和触发事件）

### 第二幕：发展
（冲突升级和角色成长）

### 第三幕：高潮与结局
（最终对决和故事收束）

## 章节规划

### 第一章：[章节名]
- 目标：
- 主要情节：
- 章末钩子：

（后续章节...）

## 伏笔与呼应
- 伏笔1 → 呼应位置
- 伏笔2 → 呼应位置
```

### 质量标准

- 情节要有因果逻辑，事件之间有合理关联
- 节奏张弛有度，紧张与舒缓交替
- 每章都有明确目的，推动情节或揭示角色
- 预留伏笔空间，前后呼应
- 角色弧光清晰，有成长变化

### 爽点分布规划

**爽点层级**：
- 基础爽点：打脸/逆袭/碾压（每章至少1个）
- 升级爽点：基础+意外收获+连锁反应（每3-5章）
- 极致爽点：多重打脸+真相揭露+情感宣泄+格局突破（每卷高潮）

**爽点公式**：
- 铺垫压抑 → 蓄势待发 → 爆发释放 → 余韵回味
- 压抑越深，爽感越强

### 节奏循环模板

**升级打怪循环**（5-7章一循环）：
1. 新环境/新挑战出现
2. 遭遇困难/压制
3. 寻找突破口
4. 逆袭/战胜
5. 收获/升级
6. 新的危机预兆

**螺旋上升结构**：
- 每10章一个小循环（绝境→突破）
- 每50章一个大循环（覆灭→重生）
- 整体呈现"否极泰来"的太极循环

### 钩子设计

**章末钩子类型**：
- 悬念钩：留下未解之谜
- 危机钩：新的威胁出现
- 反转钩：意外信息揭露
- 期待钩：预告即将到来的精彩

**长线钩子**：
- 身世之谜：主角的真实身份
- 宝物线索：强大道具的下落
- 宿敌伏笔：终极反派的暗示
- 预言/诅咒：命运的牵引

### 上下文利用

- 参考已有角色设定，确保情节能展现角色特点
- 参考世界观设定，确保情节符合世界规则
- 如有已写章节，确保大纲与已有内容衔接
