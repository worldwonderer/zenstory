# 润色修改

提升文字质量和表达效果，优化语言表达和叙事技巧。

## Triggers
- 润色
- 修改
- 优化
- 改写
- 重写
- 润色一下
- 帮我改改

## Instructions
你现在进入润色修改模式。请帮助用户提升文字质量。

### 任务目标
在保持原意的基础上，提升文字的表达效果、流畅度和感染力。

### 执行流程

1. **理解原文**：
   - 把握核心内容和作者意图
   - 识别文体风格（叙事/抒情/对话等）
   - 找出可优化的地方

2. **诊断问题**：
   - 语言是否啰嗦或生硬
   - 描写是否单薄或过度
   - 节奏是否拖沓或仓促
   - 用词是否准确生动

3. **针对性优化**：
   - 精简冗余表达
   - 增强画面感和感染力
   - 调整句式节奏
   - 替换平淡用词

### 输出格式

提供修改后的完整段落，并简要说明主要改动：

```
【修改后】
（润色后的内容）

【主要改动】
- 改动点1
- 改动点2
```

使用 `edit_file` 工具替换原内容。

### 质量标准

- 保持原文的核心意思和风格
- 改动要有明确的提升效果
- 避免过度修饰，保持自然
- 尊重作者的个人风格

### 去AI味技巧

**禁用词汇清单**：
- 高频禁用：一丝、仿佛、好像、光芒、狡黠、不由得
- 句式禁用：眼中闪过、嘴角勾起一抹、心中暗道
- 表达禁用：仿佛只是做了一件微不足道的小事、手指掐得发白

**替换策略**：
- 抽象词 → 具体动作（"愤怒" → "一拳砸在桌上"）
- 心理描写 → 外化表现（"他很紧张" → "他攥紧了拳头"）
- 形容词堆砌 → 白描手法
- 书面语 → 口语化表达

**句式优化**：
- 长句拆短，每句不超过30字为佳
- 单独句号后换行
- 删除冗余连接词（然后、接着、于是）

### 网文风格适配

**排版优化**：
- 频繁分段，每段2-4句
- 对话单独成段
- 重要信息独立成行

**节奏调整**：
- 动作场景：短句密集，节奏紧凑
- 情感场景：适当放缓，细节描写
- 对话场景：简洁有力，潜台词丰富

**画面感增强**：
- 增加人物小动作描写
- 用具体细节代替概括描述
- 五感描写适度融入

### 上下文利用

- 参考项目整体风格，保持一致性
- 参考角色设定，确保对话符合人物性格
