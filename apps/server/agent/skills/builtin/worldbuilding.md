# 世界观设定

帮助用户构建故事的世界观和背景设定。

## Triggers
- 世界观
- 世界设定
- 背景设定
- 创建世界
- 设定集

## Instructions
你现在进入世界观设定模式。

### 任务目标
构建完整、自洽的故事世界观，为角色和情节提供可信的舞台背景。

### 执行流程

1. **确定世界类型**：
   - 现实世界（哪个时代、哪个地区）
   - 架空世界（奇幻/科幻/末世等）
   - 现实与虚构结合

2. **设计核心规则**：
   - 这个世界与现实的关键差异
   - 特殊力量体系（魔法/科技/异能）
   - 规则的限制和代价

3. **构建社会结构**：
   - 政治体系和权力分布
   - 经济模式和阶层划分
   - 文化习俗和价值观念

4. **规划地理环境**：
   - 主要地点和场景
   - 地点之间的关系
   - 环境对故事的影响

5. **设定历史背景**：
   - 影响当前的重大历史事件
   - 各势力的形成和演变

### 输出格式

使用 `create_file` 工具创建设定文件，文件类型为 `lore`，格式如下：

```
# [世界名称] 世界观设定

## 世界概述
（一段话概括这个世界的核心特点）

## 核心规则
### 特殊体系
（魔法/科技/异能等的运作方式）

### 规则限制
（使用代价和限制条件）

## 社会结构
### 政治体系
（权力分布和统治方式）

### 阶层划分
（社会阶层和流动性）

## 主要地点
### [地点名称]
- 特点：
- 重要性：

## 历史大事记
- [年代] [事件]
```

### 质量标准

- 设定要内部自洽，不能自相矛盾
- 规则要有限制，避免万能设定
- 留有探索空间，不必事无巨细
- 设定要服务于故事，不是为设定而设定
- 考虑设定对角色和情节的影响

### 上下文利用

- 参考已有角色，确保世界观能容纳这些角色
- 参考大纲，确保世界观支持计划中的情节
- 如有已写内容，确保设定与已有描述一致
