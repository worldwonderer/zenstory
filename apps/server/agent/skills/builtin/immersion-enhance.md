# 代入感强化

通过五感描写和情绪渲染增强读者的沉浸体验。

## Triggers
- 代入感
- 沉浸感
- 五感描写
- 增强代入
- 画面感
- 氛围感

## Instructions
你现在进入代入感强化模式。请帮助用户增强文字的沉浸体验。

### 任务目标
通过五感描写、情绪渲染和细节刻画，让读者身临其境，与角色产生共鸣。

### 执行流程

1. **分析当前内容**：
   - 识别缺乏代入感的段落
   - 找出可以强化的感官点
   - 确定需要渲染的情绪

2. **五感强化**：
   - 补充缺失的感官描写
   - 用具体细节替代抽象描述
   - 平衡各感官的比例

3. **情绪渲染**：
   - 通过动作外化内心
   - 用环境烘托氛围
   - 让读者感同身受

### 五感描写技巧

**视觉**（占比40%）：
- 色彩：冷暖色调营造氛围
- 光影：明暗对比增加层次
- 动态：动作细节增强画面感
- 示例："夕阳将他的影子拉得老长，血红的光芒染红了半边天。"

**听觉**（占比25%）：
- 环境音：风声、雨声、人声
- 语气：说话的音调、节奏
- 特殊音效：脚步声、心跳声
- 示例："寂静中，只有自己的心跳声，一下一下，震得耳膜发疼。"

**触觉**（占比20%）：
- 温度：冷热感受
- 质感：粗糙、光滑、柔软
- 痛感：疼痛的具体描述
- 示例："冰冷的刀锋贴上脖颈，寒意直透骨髓。"

**嗅觉**（占比10%）：
- 气味：花香、血腥、腐臭
- 空气：潮湿、干燥、清新
- 示例："空气中弥漫着铁锈般的血腥味。"

**味觉**（占比5%）：
- 口感：苦涩、甘甜
- 特殊场景使用
- 示例："嘴里泛起一股铁锈味，是血。"

### 情绪外化技巧

**用动作代替心理**：
- "他很紧张" → "他攥紧了拳头，指节泛白"
- "她很愤怒" → "她一把掀翻了桌子"
- "他很害怕" → "他的腿在发抖，几乎站不稳"

**用环境烘托情绪**：
- 紧张：空气凝固、时间变慢
- 悲伤：阴雨、昏暗、寂静
- 喜悦：阳光、明亮、热闹

### 输出格式

```
【原文】
（需要强化的段落）

【强化后】
（增加五感和情绪描写的版本）

【强化要点】
- 增加了XX感官描写
- 通过XX动作外化了XX情绪
```

### 质量标准

- 五感描写要自然融入，不能生硬堆砌
- 情绪渲染要适度，避免过度煽情
- 细节要服务于情节，不能喧宾夺主
- 保持原文风格，不要过度修饰

### 上下文利用

- 参考角色设定，确保描写符合人物特点
- 参考世界观设定，确保细节符合背景
