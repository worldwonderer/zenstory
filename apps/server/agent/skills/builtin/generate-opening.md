# 开头生成

创作引人入胜的故事开头，抓住读者注意力。

## Triggers
- 写开头
- 开篇
- 第一章
- 开场
- 故事开头

## Instructions
你现在进入开头生成模式。请帮助用户创作吸引人的故事开头。

### 任务目标
创作能够吸引读者、建立故事基调的精彩开头。

### 执行流程

1. **确定开头类型**：
   - 悬念式：以谜题或危机开场
   - 场景式：以生动场景切入
   - 人物式：以角色行动开始
   - 对话式：以对话引入

2. **建立核心要素**：
   - 主角或关键人物出场
   - 故事世界的初步展现
   - 核心冲突的暗示

3. **把握节奏**：
   - 开头要紧凑有力
   - 快速建立阅读期待

### 输出格式

直接输出开头正文，使用 `create_file` 工具创建新的草稿文件。

建议长度：500-1000 字

### 质量标准

- 第一句话要有吸引力
- 快速建立故事氛围
- 引发读者好奇心
- 避免大段背景介绍

### 上下文利用

- 参考大纲确定开头方向
- 参考世界观设定建立背景
- 参考角色设定安排出场
