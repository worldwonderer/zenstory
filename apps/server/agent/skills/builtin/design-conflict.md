# 冲突设计

设计戏剧冲突和情节转折，让故事更有张力。

## Triggers
- 冲突
- 矛盾
- 转折
- 高潮
- 设计冲突

## Instructions
你现在进入冲突设计模式。请帮助用户设计有张力的戏剧冲突。

### 任务目标
设计能推动情节发展、揭示人物性格的戏剧冲突。

### 执行流程

1. **分析现有情节**：
   - 当前故事发展到什么阶段
   - 主要角色的目标和动机
   - 已有的矛盾和伏笔

2. **选择冲突类型**：
   - 人与人：角色间的对立
   - 人与环境：外部困境
   - 人与自我：内心挣扎

3. **设计冲突要素**：
   - 冲突的起因
   - 升级的过程
   - 爆发的高潮
   - 解决或转化

### 输出格式

```
【冲突设计】

类型：（人与人/人与环境/人与自我）

起因：
（冲突如何产生）

发展：
（冲突如何升级）

高潮：
（冲突的爆发点）

影响：
（对人物和情节的影响）
```

### 质量标准

- 冲突要有合理的动机支撑
- 冲突要能推动情节发展
- 冲突要能揭示或改变人物
- 避免为冲突而冲突

### 冲突密度控制

**冲突频率**：
- 95%的章节应包含有效冲突
- 每1000字至少有1个冲突点或矛盾升级
- 冲突要层层加码，不断升级

**冲突层次**：
- 表层冲突：言语对抗、肢体冲突
- 中层冲突：利益争夺、立场对立
- 深层冲突：价值观碰撞、信念动摇

**冲突类型组合**：
- 身份反转与认知颠覆
- 暴力冲突与生死危机
- 道德困境与极端背叛
- 高密度信息冲突

### 反转技术

**反转频率**：
- 小反转：每3-5章（预期偏差）
- 中反转：每20-30章（认知颠覆）
- 大反转：每100章（世界观重塑）

**反转类型**：
- 身份反转：敌人变盟友、盟友变敌人
- 真相反转：已知信息被颠覆
- 结果反转：预期结局被打破

**反转原则**：
- 反转必须有前期铺垫（伏笔）
- 反转后要合情合理
- 反转要带来新的冲突或发展

### 三重拉扯设计

**情绪拉扯**：
- 绝望↔希望的极限摆动
- 让读者情绪跟随角色起伏

**预期拉扯**：
- 设定期待→打破期待→超越期待
- 给读者惊喜而非失望

**节奏拉扯**：
- 压迫→喘息→更大危机
- 张弛有度，避免疲劳

### 上下文利用

- 参考角色设定，利用性格差异制造冲突
- 参考大纲，确保冲突符合整体规划
