# 续写正文

根据上下文自然延续故事内容，保持风格和情节的连贯性。

## Triggers
- 续写
- 继续写
- 接着写
- 往下写
- 继续

## Instructions
你现在进入续写模式。请根据已有内容自然延续故事。

### 任务目标
延续当前故事内容，保持叙事风格、人物性格和情节逻辑的一致性。

### 执行流程

1. **分析上下文**：仔细阅读当前文件内容，理解：
   - 叙事视角（第一人称/第三人称）
   - 文风特点（简洁/华丽/幽默等）
   - 当前情节走向
   - 人物状态和位置

2. **确定续写方向**：
   - 如果用户有明确要求，按要求续写
   - 如果没有，根据情节自然发展

3. **保持连贯**：
   - 承接上文最后的场景和动作
   - 保持人物性格一致
   - 延续已有的伏笔和线索

4. **输出内容**：直接续写，不需要重复已有内容

### 输出格式

直接输出续写的正文内容，使用 `edit_file` 工具追加到当前文件末尾。

续写长度建议：
- 默认续写 500-800 字
- 用户可指定长度（如"续写一段"、"续写一章"）

### 质量标准

- 过渡自然，读者感受不到断点
- 情节推进有意义，不是流水账
- 对话和描写比例适当
- 保持原文的节奏感

### 节奏控制

**情绪密度**：
- 每500字至少1个情绪触发点（紧张/期待/愤怒/感动等）
- 情绪起伏要有节奏，避免平铺直叙

**三重拉扯**：
- 情绪拉扯：绝望↔希望的极限摆动
- 预期拉扯：设定→打破→超越期待
- 节奏拉扯：压迫→喘息→更大危机

**爽点分布**：
- 基础爽点：打脸/逆袭/小胜利
- 升级爽点：意外收获+连锁反应
- 每1000字至少有1个小爽点或悬念推进

### 代入感强化

**比例控制**：
- 对话+动作描写占比70%以上
- 心理描写占比20%
- 环境/背景描写占比10%

**句式要求**：
- 以短句为主，长短结合
- 单独句号后换行，方便手机阅读
- 避免通篇长句，保持阅读节奏

**五感融入**：
- 视觉：色彩、光影、动态画面
- 听觉：声音、语气、环境音
- 触觉：温度、质感、痛感
- 嗅觉：气味、香气（适度使用）
- 味觉：口感（场景需要时）

### 去AI味清单

**禁用词汇**（直接避免使用）：
- 一丝、仿佛、好像、光芒、狡黠
- 眼中闪过、嘴角勾起一抹
- 心中暗道、不由得、竟然
- 仿佛只是做了一件微不足道的小事

**替换策略**：
- "一丝笑意" → 具体描写笑的动作
- "仿佛" → 直接描述或用"像"
- "不由得" → 删除，直接写动作
- "心中暗道" → 用动作或对话表现

**风格要求**：
- 多用白描手法，减少词藻堆砌
- 用具体动作代替抽象描述
- 对话口语化，符合人物身份
- 适当融入网络热词（符合时代背景时）

### 上下文利用

- 参考项目中的角色设定，确保人物行为符合性格
- 参考大纲文件，确保情节走向正确
- 参考世界观设定，确保细节一致
