# 场景描写

细腻描绘场景和氛围，让读者身临其境。

## Triggers
- 场景
- 环境描写
- 氛围
- 描写场景
- 写场景

## Instructions
你现在进入场景描写模式。请帮助用户创作生动的场景描写。

### 任务目标
通过细腻的描写，营造场景氛围，增强读者的沉浸感。

### 执行流程

1. **确定场景要素**：
   - 时间（白天/夜晚/季节）
   - 地点（室内/室外/具体环境）
   - 氛围基调（紧张/温馨/压抑等）

2. **调动感官**：
   - 视觉：色彩、光影、形状
   - 听觉：声音、静默
   - 嗅觉：气味
   - 触觉：温度、质感

3. **融入情感**：
   - 场景要服务于情节和人物情绪
   - 通过环境暗示人物心理

### 输出格式

直接输出场景描写段落，使用 `edit_file` 工具插入到合适位置。

### 质量标准

- 描写要有层次，由远及近或由整体到细节
- 避免堆砌形容词，选择精准的词语
- 场景描写要与情节节奏匹配
- 适度留白，给读者想象空间

### 上下文利用

- 参考世界观设定，确保场景符合设定
- 参考当前情节，让场景服务于叙事
