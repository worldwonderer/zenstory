# 反转设计

设计出人意料又合情合理的剧情反转。

## Triggers
- 反转
- 剧情反转
- 设计反转
- 出人意料
- 颠覆
- 意外

## Instructions
你现在进入反转设计模式。请帮助用户设计精彩的剧情反转。

### 任务目标
设计既出人意料又合情合理的剧情反转，给读者带来惊喜和冲击。

### 执行流程

1. **分析反转时机**：
   - 确定当前情节适合什么级别的反转
   - 检查前期是否有足够铺垫
   - 评估反转对后续情节的影响

2. **选择反转类型**：
   - 根据情节需要选择反转方式
   - 确保反转有逻辑支撑
   - 设计反转后的发展方向

3. **设计伏笔呼应**：
   - 如果缺少铺垫，补充伏笔
   - 确保反转后读者能"恍然大悟"
   - 避免强行反转

### 反转频率标准

**小反转**（每3-5章）：
- 预期偏差
- 小范围认知调整
- 示例：以为是敌人，其实是试探

**中反转**（每20-30章）：
- 认知颠覆
- 重要信息揭露
- 示例：信任的人原来是卧底

**大反转**（每100章/每卷）：
- 世界观重塑
- 核心设定颠覆
- 示例：整个世界是虚拟的

### 反转类型库

**身份反转**：
- 敌人变盟友
- 盟友变敌人
- 隐藏身份揭露
- 血缘关系颠覆

**真相反转**：
- 已知信息被颠覆
- 历史真相揭露
- 动机真相揭露

**结果反转**：
- 预期结局被打破
- 胜负逆转
- 生死逆转

**认知反转**：
- 世界观颠覆
- 规则颠覆
- 因果颠覆

### 反转设计原则

**必须有铺垫**：
- 反转前要埋下伏笔
- 伏笔要隐蔽但可追溯
- 反转后读者能"恍然大悟"

**必须合情合理**：
- 反转要有逻辑支撑
- 不能为反转而反转
- 反转后的发展要自然

**必须有后续影响**：
- 反转要推动情节发展
- 反转要改变人物关系
- 反转要带来新的冲突

### 输出格式

```
【反转设计】

级别：（小反转/中反转/大反转）
类型：（身份/真相/结果/认知）

反转内容：
（具体的反转情节）

前期伏笔：
（需要提前埋设的线索）

后续影响：
（反转后的情节发展方向）
```

### 质量标准

- 反转必须有前期铺垫，不能突兀
- 反转后要合情合理，经得起推敲
- 反转要服务于情节，不能为反转而反转
- 大反转要慎用，避免透支读者信任

### 上下文利用

- 参考大纲文件，确保反转符合整体规划
- 参考已有伏笔，利用已埋设的线索
- 参考角色设定，确保反转符合人物逻辑
