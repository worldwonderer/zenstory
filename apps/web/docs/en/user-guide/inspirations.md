# Inspirations Library

The Inspirations Library helps you turn rough ideas into actionable writing plans faster.

## Where to Find It

- Dashboard sidebar: **Inspirations**
- Dashboard home: **Featured Inspirations** → “View All”

## Core Features

### 1) Browse and Filter

- Filter by category and tags
- Search by title and summary keywords
- Explore featured and recent picks

### 2) View Inspiration Details

An inspiration detail page typically includes:

- Title and core hook
- Genre/tags
- Setting and character dynamics
- Core conflict and expansion directions

### 3) Copy into a Project

From the detail page, you can copy content directly into your current project, usually into:

- Outline files (story skeleton)
- Character files (character prototypes)
- Lore files (world rules and settings)

After copying, do a quick rewrite to keep your voice and avoid template-like output.

## Recommended Workflow

1. Pick one inspiration that matches your current writing goal  
2. Copy it into your project (outline/character/lore)  
3. In AI chat, give explicit follow-up prompts, for example:  
   - “Give me 3 possible plot directions based on this inspiration.”  
   - “Keep the same conflict, but rewrite in a suspense tone.”  
4. Lock one direction, then expand chapter by chapter

## FAQ

### What’s the difference between Inspirations and Materials?

- **Inspirations**: creative templates and starting directions  
- **Materials**: structured references extracted from uploaded novels

Use them together: Inspirations for direction, Materials for depth.
