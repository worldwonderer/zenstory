# 高效写作工作流

本文档将为您提供一套经过实践验证的高效写作工作流程，帮助您充分发挥zenstory写作工作台的优势，提升创作效率和作品质量。

## 推荐的创作流程

zenstory的工作台采用三阶段创作流程，每个阶段都有明确的工具和AI辅助策略，让您的创作更加系统化和高效。

### 阶段一：构思与规划

![项目初始界面，展示文件树中包含AI记忆、世界观、角色卡、大纲文件的完整结构](/docs-images/advanced/workflow-tips.png)

这是整个创作的基础阶段。良好的规划能够显著减少后续创作中的返工，让AI更精准地理解您的创作意图。

**1. 创建项目**
- 在首页点击「新建项目」，为您的作品命名
- 填写项目简介，这将帮助AI理解整体创作方向
- 选择合适的模板（网文连载、短篇故事等）

**2. 填写AI记忆**
- AI记忆是全局上下文的核心，所有对话都会参考这里的信息
- 包含：作品类型、写作风格偏好、目标读者群体、核心主题等
- 建议：越详细越好，可以随时更新
- 示例：*"这是一部都市玄幻小说，风格轻松幽默，目标读者是18-25岁的年轻读者，擅长使用网络流行语和梗"*

**3. 创建世界观文件**
- 文件类型选择「设定」(lore)
- 填写关键设定：魔法体系、势力分布、地理环境、历史背景等
- 分门别类：建议创建多个设定文件，如「魔法体系」「主要势力」「世界地图」
- 与AI协作：可以让AI帮助您完善世界观细节，发现潜在的逻辑漏洞

**4. 创建主要角色卡**
- 文件类型选择「角色」(character)
- 核心信息：姓名、年龄、外貌、性格、背景、动机、目标
- 进阶信息：人物关系、口头禅、习惯动作、成长弧线
- 建议：主角和重要配角都需要详细的角色卡
- AI辅助：向AI描述角色雏形，让其帮助丰富人物形象

**5. 生成分章大纲**
- 文件类型选择「大纲」(outline)
- 利用已有的世界观和角色卡，与AI协作生成章节大纲
- 明确每章的：主要事件、场景、出场人物、情节推进目标
- 大纲层级：卷 > 章 > 节（如果需要）
- 灵活调整：大纲不是死的，后续可以根据创作进展调整

**时间建议**：规划阶段通常需要1-3天，占整体创作时间的10-15%，但能节省后续30%以上的修改时间。

### 阶段二：章节创作

这是核心创作阶段，zenstory的三栏布局设计让您可以在同一界面完成所有工作，无需切换窗口。

**1. 选择章节大纲**
- 从左侧文件树中打开要创作的章节大纲
- 在中间编辑器中查看本章的目标和要点
- 将相关角色卡和设定文件也在后台打开（AI会自动引用）

**2. 与AI讨论情节**
- 在右侧AI对话面板中开始对话
- 告诉AI您想要创作的场景："我想写第3章，主角在古玩市场发现了神秘玉佩"
- 让AI提供情节建议、场景描写、对话设计等
- 逐步细化：不要一次性要求AI生成完整章节，而是分段讨论

**3. 生成初稿**
- 当您对情节方向满意后，请AI生成正式文本
- 使用提示词示例：*"请根据我们刚才的讨论，写出第3章的开场部分（约1000字），场景是古玩市场，氛围要悬疑"*
- AI会参考您的AI记忆、角色卡、世界观设定，保持文风一致性

**4. 人工修改润色**
- 将AI生成的内容复制到编辑器中
- 进行人工修改：调整措辞、增加细节、修正不符合设定的部分
- 添加您的个人风格和创意火花
- zenstory支持实时自动保存，无需担心丢失

**5. 版本保存**
- 完成一版满意的稿件后，系统会自动创建版本快照
- 也可以手动创建重要版本（如"第一稿""精修版"）
- 版本历史让您可以随时回溯和对比

**时间建议**：单个章节创作通常需要2-4小时（3000-5000字），AI辅助可以提升50%以上的效率。

### 阶段三：修订完善

这是提升作品质量的关键阶段，zenstory提供了多种工具帮助您精细化打磨作品。

**1. 版本对比**
- 选择任意两个版本进行对比
- 差异会高亮显示：新增内容（绿色）、删除内容（红色）、修改内容（黄色）
- 快速识别修改轨迹，做出更好的创作决策

**2. AI辅助润色**
- 选中需要润色的段落
- 向AI提出具体要求：*"请帮我润色这段描写，让它更有画面感和紧张氛围"*
- AI会保持您的基本文风，优化表达和节奏
- 可以多次迭代，直到满意为止

**3. 一致性检查**
- 让AI检查章节与角色卡、世界观设定的一致性
- 发现潜在问题：如角色能力前后矛盾、时间线错乱等
- 示例提示词：*"请检查这一章是否与我之前设定的魔法体系一致"*

**时间建议**：修订阶段通常占创作时间的20-30%，是提升作品质量的关键投资。

## 文件组织建议

良好的文件组织能让您快速找到所需资料，提升创作流畅度。zenstory支持树状文件结构，您可以灵活组织。

### 按卷分文件夹

**推荐结构**：
```
├── AI记忆.md
├── 世界观/
│   ├── 魔法体系.md
│   ├── 主要势力.md
│   └── 世界地图.md
├── 角色/
│   ├── 主角/
│   │   └── 张三.md
│   └── 配角/
│       ├── 李四.md
│       └── 王五.md
├── 第一卷/
│   ├── 卷大纲.md
│   ├── 第1章/
│   │   ├── 章节大纲.md
│   │   └── 初稿.md
│   └── 第2章/
│       ├── 章节大纲.md
│       └── 初稿.md
└── 素材库/
    ├── 灵感片段.md
    └── 待用设定.md
```

**组织技巧**：
- 每卷一个文件夹，章节作为子文件
- 角色和世界观单独成文件夹，便于跨章节引用
- 使用命名规范：如"第1章_初遇.md"比"1.md"更清晰
- 定期整理：删除废弃文件，归档已完成章节

## AI协作技巧

与AI的有效协作是提升创作效率的关键。以下技巧能帮助您获得更好的AI输出。

### 分步骤提问

**错误示范**：
*"请帮我写第3章，主角遇到女主角，两人产生误会，最后解开误会，字数3000字"*

**正确示范**：
1. 先问：*"第3章我想让主角和女主角初次相遇，请给我三个场景建议，每个场景100字简介"*
2. 选择场景后问：*"我选择第一个场景，请帮我写详细的开场描写（500字）"*
3. 继续问：*"接下来两人发生对话，请帮我写这段对话，体现女主角的傲娇性格"*
4. 逐步推进，直到章节完成

**为什么有效**：
- AI更容易在具体、明确的指令下产出高质量内容
- 您可以及时调整方向，避免大量返工
- 更好地控制节奏和细节

### 提供足够上下文

**技巧**：
- 明确引用相关文件：*"请参考【张三角色卡】中的性格设定"*
- 提供前文摘要：*"上一章主角刚获得了神秘玉佩"*
- 说明创作目标：*"这段要营造紧张氛围，为后续高潮铺垫"*

zenstory的AI会自动读取相关文件，但您的明确提示能让AI更精准地定位关键信息。

### 善用确认机制

- **检查AI输出**：不要盲目接受AI生成的内容，检查是否符合设定和逻辑
- **及时修正**：发现问题时，立即告诉AI修正，如*"这个对话不符合张三的性格，请重新写"*
- **要求解释**：可以让AI解释其创作思路，*"为什么选择这个场景？如何推进情节？"*

AI是您的协作伙伴，而不是代笔者。您的判断和审美仍然是作品质量的核心保障。

## 备份策略

创作安全至关重要，以下是多层保障策略：

### 定期导出

- 使用右上角的「导出」功能
- 支持格式：纯文本(.txt)（当前版本）
- 建议：每完成一章就导出一次，保存在本地或云盘

### 利用版本历史

- zenstory自动保存所有版本
- 可以随时回溯到任意历史版本
- 重要节点手动创建标记版本（如"第一卷完成"）

### 多处保存

- **云端**：zenstory自动同步到云端
- **本地**：定期导出到本地硬盘
- **备份**：使用云盘（如iCloud、Google Drive）同步导出文件
- **冷备**：重要作品定期刻录光盘或保存到移动硬盘

**黄金法则**：数据至少存在3个地方，其中至少1个是离线的。

## 常见工作流模板

不同类型的创作有不同的最佳实践，以下是两种常见场景的推荐工作流。

### 网文连载工作流

**特点**：追求快速产出、灵活调整、保持更新节奏

**推荐流程**：
1. **快速规划**：只规划大致世界观和主线，细节在连载中补充
2. **批量大纲**：一次规划10-20章大纲，避免卡文
3. **日更节奏**：
   - 上午：与AI讨论今日章节要点（15分钟）
   - 下午：AI辅助生成初稿（1-2小时）
   - 晚上：人工润色修改（1小时）
4. **灵活调整**：根据读者反馈及时调整后续情节
5. **周末复盘**：每周回顾整体进度，调整下周大纲

**效率技巧**：
- 使用AI生成过渡段落和日常对话，节省时间
- 建立角色对话模板库，保持人物声音一致
- 善用"继续写作"功能，让AI基于前文自动续写

### 短篇创作工作流

**特点**：追求完整性、精细打磨、一次成型

**推荐流程**：
1. **完整规划**：详细规划所有角色、场景、情节转折
2. **分场景创作**：
   - 第1天：与AI协作完成场景规划和开篇（2000字）
   - 第2天：发展部分，推进情节（3000字）
   - 第3天：高潮和结尾（3000字）
3. **整体打磨**：
   - 第4天：通读全文，修改逻辑问题
   - 第5天：与AI协作进行语言润色
   - 第6天：最终检查和格式调整
4. **外部反馈**：分享给beta读者，收集反馈后精修

**质量技巧**：
- 充分利用版本对比功能，比较不同修改版本
- 让AI从读者角度评价作品："这段是否过于突兀？"
- 多次迭代润色，直到每个句子都恰到好处

## 结语

高效的写作工作流不是一成不变的教条，而是您在实践中不断优化的系统。zenstory提供的工具和AI协作能力，能够适应各种创作风格和习惯。

**记住核心原则**：
1. **先规划后创作**：好的规划是高效的基础
2. **分步骤与AI协作**：逐步细化比一次性生成更有效
3. **人工把关质量**：AI是助手，您是创作者
4. **安全第一**：多重备份，永不丢失作品
