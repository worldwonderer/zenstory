# 订阅与权益

本页介绍 zenstory 的套餐查看、配额理解、升级路径，以及积分/邀请码相关入口。

## 入口位置

- 仪表盘侧边栏：**订阅权益**
- 公开页面：**Pricing / 价格页**

## 你可以在这里做什么

## 1) 查看当前套餐

- 当前套餐名称
- 套餐状态（是否已开通）
- 到期/续费相关提示（如有）

## 2) 查看配额使用

常见会展示的维度包括：

- 可创作体量
- Agent 深度任务次数
- 可并行项目数
- 上下文容量
- 导出能力

不同套餐会对应不同上限与优先级。

## 3) 对比套餐差异

在套餐对比区可查看：

- 月付 / 年付切换
- 关键能力差异（可切换“仅看差异”）
- 适用人群建议（轻度创作 / 日更 / 团队协作）

## 4) 升级与兑换

你可以通过两种方式获得更高权益：

- 直接升级套餐
- 使用兑换码开通权益

> 建议：先按你的创作目标（更新频率、项目数量、改稿强度）选套餐，再比较价格周期。

## 关于积分与邀请码

- 邀请成功可获得积分奖励
- 被邀请人通常可获得试用权益
- 积分相关活动和明细可在应用内对应页面查看

## 常见问题

### 为什么我看不到某些高级能力？

通常是当前套餐未包含该能力。请到「订阅权益」页查看能力对比与可升级方案。

### 导出能力受套餐影响吗？

会。当前版本导出以 `.txt` 为主，具体以你套餐中显示的能力为准。
