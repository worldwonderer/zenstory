# AI创作助手

## 什么是AI创作助手?

AI创作助手是zenstory的核心功能,它就像一个懂你的写作伙伴。它能够理解你的故事背景、角色设定和写作风格,为你提供创意灵感、内容生成和智能建议。无论是构思大纲、丰富角色、还是撰写正文,AI都能成为你得力的助手。

与传统写作工具不同,AI创作助手会主动理解你的项目上下文。当你打开某个大纲或草稿时,AI会自动获取相关内容;当你选中一段文字时,AI能基于这段文字进行续写或修改。这种深度理解让创作更加流畅自然。

## 开始对话

### 输入创作需求

![聊天输入框位于界面右侧底部](/docs-images/user-guide/ai-assistant.png)

在右侧聊天面板底部的输入框中,输入你想要AI帮助的内容。

**示例输入:**
- "帮我为女主角设计一个跌宕起伏的身世背景"
- "根据当前大纲,续写第三章的开头"
- "优化这段对话,让角色的性格更鲜明"

输入框支持多行文本,你可以详细描述需求。按 `Enter` 发送消息,按 `Shift + Enter` 换行。

### 使用建议提示

如果你不知道从何开始,可以点击输入框上方的建议提示。这些提示会根据你的项目状态动态生成,帮助你快速发起对话。

**提示类型:**
- **新项目引导** - 当项目刚创建时,提供创建大纲、角色等基础内容的建议
- **AI智能建议** - 根据你最近的对话内容,AI会生成相关的后续建议

**操作方式:**
- 点击任意建议气泡,内容会自动填入输入框
- 点击刷新按钮获取新的建议
- 按 `Tab` 键快速采纳第一条建议

### 语音输入

如果你更习惯口述想法,可以使用语音输入功能。

**PC端操作:**
1. 点击麦克风按钮开始录音
2. 对着麦克风说话(最长55秒)
3. 再次点击按钮停止录音,AI会自动识别并转成文字

**移动端操作:**
1. 长按麦克风按钮开始录音
2. 保持按住并说话
3. 松开按钮停止录音并识别

**注意事项:**
- 需要浏览器授权麦克风权限
- 右键点击(或长按后滑开)可取消录音
- 识别结果会追加到输入框现有内容后

## AI的响应

### 思考过程

当AI处理复杂任务时,会先展示思考过程。这部分内容以半透明样式显示,默认展开,你可以点击标题栏折叠它。

**思考过程包括:**
- AI对任务的理解
- 计划采取的步骤
- 考虑到的上下文因素

思考过程帮助你了解AI的推理逻辑,也让创作过程更加透明。

### 创作结果

AI的主要回复会以清晰的格式展示,支持:
- **Markdown格式** - 标题、列表、加粗等
- **代码块** - 如果涉及格式示例
- **实时流式输出** - 你能看到AI逐字生成内容

### 工具操作

当AI执行具体操作时(如创建文件、编辑内容),会显示工具结果卡片:

**创建文件:**
```
✓ 已创建 大纲「第一章」
```

**编辑文件:**
```
✓ 编辑完成 「第三章」
[查看编辑详情 ▼]
```

点击"查看编辑详情"可以看到:
- 被删除的内容(红色背景)
- 新增的内容(绿色背景)
- 操作类型(替换/插入/追加/删除)

**查询文件:**
```
✓ 找到 3 个相关文件
- 角色「李明」
- 设定「世界观-修仙体系」
- 大纲「第一卷」
```

## 与AI协作

### 确认与拒绝

AI的某些操作可能需要你的确认。当AI不确定该如何继续时,会暂停并询问你的意见:

```
⚠ 等待你的回复
AI需要更多确认才能继续
```

这时你可以:
- 提供更多细节让AI继续
- 修改之前的指令
- 发送"继续"让AI按自己的判断进行

### 撤销操作

如果AI编辑了文件但你不太满意,可以撤销这次修改:

1. 在工具结果卡片中找到"撤销"按钮
2. 点击后,AI的修改会被回滚到编辑前的版本
3. 撤销后你可以要求AI重新生成

**注意:** 撤销功能仅对最近的编辑有效,如果之后又有新的编辑,之前的编辑可能无法撤销。

### 多轮对话

你可以和AI进行多轮对话,逐步完善内容:

**示例流程:**
```
你: 帮我设计一个反派角色
AI: [生成角色设定]
你: 让他更复杂一些,给一个悲惨的童年
AI: [修改并补充背景]
你: 现在基于这个角色,写一段他的出场
AI: [生成正文片段]
```

AI会记住对话历史,理解上下文关系。你可以随时追问、修改或要求重新生成。

## 上下文管理

AI会自动获取和使用项目的上下文信息,但你也可以主动管理这些上下文。

### 当前编辑文件

当你打开某个文件(大纲、角色、设定、草稿)时,AI会自动将其作为"焦点文件":

- **自动传递:** 你发送消息时,AI会收到当前文件的内容
- **上下文理解:** AI能基于当前文件进行续写、修改或扩展

**使用场景:**
- 打开大纲 → 让AI帮你完善章节结构
- 打开角色卡 → 让AI扩展人物性格
- 打开草稿 → 让AI续写或优化正文

### 引用素材

除了当前文件,你还可以附加其他素材作为参考:

**操作方法:**
1. 在左侧文件树中右键点击文件
2. 选择"附加到对话"
3. 文件会显示在输入框上方的附件区域

**最多可附加5个文件**,AI会在处理时参考这些内容。

**适用场景:**
- 参考多个角色设定写群戏
- 基于世界观设定进行创作
- 保持前后章节的风格一致

### 引用文本

如果你想针对某段具体文字让AI处理,可以引用文本:

**操作方法:**
1. 在编辑器中选中一段文字
2. 点击"引用"按钮(或使用快捷键)
3. 选中的文字会显示在输入框上方

**最多可引用5段文本**,每段文本会记录来源文件。

**使用场景:**
- "优化这段对话的节奏"
- "把这段描写改得更生动"
- "基于这段设定进行扩展"

## 新建会话

如果你想开始一个全新的对话,可以新建会话:

1. 点击聊天面板标题栏的"+"按钮
2. 当前会话的历史消息会被清空
3. AI会开始一个新的对话上下文

**何时需要新建会话:**
- 切换到完全不同的创作任务
- 之前的对话内容干扰了新的需求
- 想要AI"忘记"之前的讨论

**注意:** 新建会话不会删除历史记录,你仍可以在对话历史中找到之前的会话。

---

## 对话历史

AI会保存你的对话历史,每次打开项目时会自动加载最近的50条消息。

**历史记录特点:**
- 跨设备同步(登录同一账号)
- 包含完整的工具调用和结果
- 支持向上滚动查看更早的消息

**如需查看完整历史:**
- 在数据库中保存了所有消息
- 可以通过后端API查询完整记录

## 使用技巧

### 1. 描述要具体

模糊的指令会让AI难以理解你的意图。

**不好的例子:**
> "写一段打斗"

**更好的例子:**
> "写一段两个剑客在雨夜的对决,要突出主角的劣势和反击的果断,风格要冷峻凌厉"

**具体描述包括:**
- 角色是谁
- 在什么场景
- 发生了什么
- 想要什么风格
- 需要什么效果

### 2. 提供足够的背景信息

虽然AI会自动获取上下文,但主动提供关键信息效果更好:

**示例:**
> "基于刚才设计的反派角色,结合世界观中的'魔修'设定,写一段他首次出场的场景。要求让读者感到威胁,但又保留一些神秘感。"

**关键信息:**
- 明确引用了之前的角色设计
- 指出了世界观中的具体设定
- 说明了想要的叙事效果

### 3. 善用引用功能

引用功能能让AI精确知道你要处理的内容:

**场景示例:**
- 在草稿中选中一段对话 → 引用 → "把这段对话改得更符合角色的阴阳怪气"
- 在设定中选中一条规则 → 引用 → "基于这个设定,设计一个相关的剧情冲突"

引用比口头描述更准确,AI能直接看到原文。

### 4. 分步骤完成复杂任务

对于复杂的创作任务,拆分成多步效果更好:

**示例:**
```
第一步: 帮我设计一个修仙世界的等级体系
第二步: 基于这个体系,创建主角的修炼进度表
第三步: 现在根据这个进度表,规划前三卷的大纲
第四步: 为第一卷写出详细的章节大纲
```

这样做的好处:
- 每一步AI都能专注于当前任务
- 你可以及时调整方向
- 最终结果更符合预期

### 5. 利用AI记忆功能

AI会记住项目的关键信息(可在"AI记忆"中查看和编辑):

**AI记忆包括:**
- 项目摘要
- 当前创作阶段
- 写作风格偏好
- 重要的备注信息

**如何利用:**
- 定期更新项目摘要,让AI了解整体进度
- 在备注中记录特殊要求
- AI会基于记忆提供更贴合的建议

### 6. 混合使用多种交互方式

最高效的工作流通常结合多种方式:

**推荐流程:**
1. 打开要编辑的文件(自动传递上下文)
2. 附加相关的设定文件(主动引用素材)
3. 选中需要优化的段落(精确引用文本)
4. 输入具体的修改要求(明确指令)

这样AI能获得最完整的上下文,给出最精准的帮助。

## 常见问题

### Q: AI会自动保存我写的内容吗?

A: 当AI创建或编辑文件时,会自动保存到数据库。每次重要修改后都会创建版本快照,你可以随时回退到之前的版本。

### Q: 为什么AI有时会"忘记"之前说的内容?

A: AI的上下文窗口有限制。如果对话非常长,较早的内容可能不在当前的上下文中。解决方法:
- 使用"引用"功能重新提供关键信息
- 新建会话,重新明确当前任务

### Q: AI的回复中断了怎么办?

A: 可能原因:
- 网络中断:刷新页面重试
- 内容太长:让AI分多次生成
- 系统超时:稍后再试

你可以发送"继续"让AI从中断处继续。

### Q: 如何让AI更好地理解我的风格?

A:
- 在项目设置中明确写作风格
- 让AI分析你已有的作品片段
- 在指令中明确风格要求(如"用古龙的风格")

### Q: AI能直接修改文件而不经过我确认吗?

A: 是的,AI默认可以直接编辑文件。但每次编辑都会:
- 创建版本快照,支持一键撤销
- 显示详细的编辑差异
- 你可以在设置中调整为"需确认"模式
