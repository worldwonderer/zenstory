# 文件类型详细说明

zenstory 采用统一的文件系统来管理您的创作内容。每种文件类型都针对特定的创作场景进行了优化，帮助您更高效地组织和管理小说的各个组成部分。

理解并正确使用文件类型，能让 AI 更好地理解您的创作意图，提供更精准的辅助。

---

## 大纲（Outline）

### 用途

大纲文件用于规划故事的整体结构和情节走向，是创作的蓝图。它帮助您：

- **章节规划**：设计故事的章节划分和情节安排
- **故事结构**：梳理主线、支线以及情节点之间的关系
- **创作指引**：为 AI 生成正文提供明确的框架和方向

### 推荐结构

大纲文件支持层级化结构，建议采用以下格式：

```
第一卷：开端
  第1章：穿越
    - 主角醒来，发现身处异世
    - 初步了解新世界的规则
    - 遇到第一个关键人物
  第2章：初遇
    - 主角与新伙伴的第一次冲突
    - 展现主角的特殊能力
    - 埋下重要伏笔

第二卷：成长
  第3章：历练
    - ...
```

### 使用建议

1. **由粗到细**：先创建整体架构大纲，再逐步细化到每一章的具体情节
2. **标注重点**：在大纲中标注关键情节、转折点和伏笔
3. **与 AI 协作**：选中大纲文件后，在 AI 对话中请求"根据这个大纲生成第1章的正文内容"
4. **持续更新**：随着创作的进行，及时更新大纲以反映实际内容的变化

## 手稿（Draft）

### 用途

手稿文件是您的核心创作空间，用于撰写小说的正文内容。这是最终输出的核心，承载着您的故事。

### 特性

- **富文本编辑**：支持文本格式化，提供流畅的写作体验
- **实时保存**：自动保存您的每一次修改，无需担心内容丢失
- **字数统计**：实时显示字数和字符数，方便控制篇幅
- **版本历史**：自动记录每次修改的历史版本，支持对比和回溯
- **AI 辅助**：支持 AI 续写、改写、扩写等功能

### 使用建议

1. **按章节组织**：每个手稿文件对应一个章节或独立的故事单元
2. **定期查看版本历史**：当对修改不满意时，可以随时回溯到之前的版本
3. **结合大纲使用**：先在大纲中规划好内容，再在手稿中展开写作
4. **善用 AI 辅助**：
   - 选中手稿文本后，让 AI 帮助润色和优化
   - 请求 AI 续写后续内容
   - 让 AI 根据角色卡和世界观设定来优化对话和场景

## 角色卡（Character）

![角色卡示例，展示角色的详细设定信息](/docs-images/reference/file-types.png)

### 用途

角色卡用于记录故事中人物的详细设定和档案。完整的角色卡能帮助 AI 保持角色性格的一致性，生成更符合人物特点的对话和行为。

### 推荐内容

一个完整的角色卡应包含以下信息：

#### 基本信息
- **姓名**：角色的全名、别名、外号
- **年龄**：实际年龄或外表年龄
- **身份**：职业、社会地位、角色定位（主角/配角/反派）

#### 外貌描述
- **容貌特征**：五官、身材、肤色等
- **穿着打扮**：常穿的服装、标志性配饰
- **特殊标记**：疤痕、纹身、胎记等

#### 性格特点
- **核心性格**：主要性格特征（如：沉稳、活泼、阴险）
- **行为习惯**：口头禅、习惯性动作
- **价值观念**：信念、底线、追求

#### 背景故事
- **出身背景**：家庭、成长环境
- **重要经历**：塑造性格的关键事件
- **人际关系**：家庭关系、朋友、敌人

#### 能力设定
- **特殊能力**：武功、魔法、技能等
- **能力来源**：天赋、修炼、奇遇等
- **能力限制**：弱点、代价

### 模板示例

```
【基本信息】
姓名：李青
年龄：24岁
身份：天剑宗内门弟子

【外貌描述】
身材高挑，剑眉星目，常着一袭青衫，腰悬长剑。
左眉骨处有一道淡淡的疤痕，是幼时练剑不慎留下。

【性格特点】
性格沉稳内敛，不喜言辞，但行事果断。
口头禅："剑在心中，人在剑外。"
重视承诺，一旦许诺必全力以赴。

【背景故事】
出身寒门，幼时被天剑宗掌门收养。
十六岁时在一次试炼中失去挚友，从此封闭内心。
与同门师妹林婉儿关系亲密，但从不承认。

【能力设定】
修炼天剑诀，已达第七层。
剑法凌厉，擅长快剑攻击。
弱点：情绪波动时剑意不稳。
```

### 使用建议

1. **主要角色全覆盖**：为所有主要角色和重要配角创建角色卡
2. **持续完善**：随着故事发展，及时更新角色的经历和关系变化
3. **关联引用**：在创建新角色时，标注与现有角色的关系
4. **AI 生成对话**：在写对话场景时，选中相关的角色卡，AI 会根据角色性格生成符合人物特点的对话

## 世界观（Lore）

### 用途

世界观文件用于构建故事的背景设定和规则体系。完善的世界观设定能让您的故事更加真实可信，也能帮助 AI 在生成内容时保持逻辑一致性。

### 推荐分类

建议将世界观按以下分类组织：

#### 世界结构
- 世界的整体架构（单一世界/多元世界）
- 地理环境、国家分布
- 重要地点的详细介绍

#### 力量体系
- 能力类型（武功/魔法/科技/异能等）
- 能力等级划分
- 修炼或提升方式
- 能力的限制和代价

#### 势力分布
- 主要国家和组织
- 门派、帮会、公司等势力
- 势力间的关系（盟友/敌对/中立）

#### 历史事件
- 重要历史节点
- 影响深远的重大事件
- 历史遗留问题

#### 特殊规则
- 世界运行的特殊法则
- 禁忌和限制
- 社会制度和文化习俗

### 使用建议

1. **分类存放**：创建"设定"文件夹，用不同的文件分类存放各类设定
2. **详略得当**：核心设定要详细，边缘设定可以简略
3. **保持一致性**：设定一旦确立，后续创作中要保持一致，避免前后矛盾
4. **及时更新**：随着故事发展补充新的设定内容
5. **AI 引用**：在与 AI 对话时，提及或选中相关的世界观设定，帮助 AI 生成符合设定的内容

## 文件类型转换

目前系统暂不支持直接转换文件类型。如果您需要更改文件类型，建议采用以下方法：

### 手动复制内容

1. 打开需要转换的文件
2. 复制文件内容
3. 在目标类型的文件夹中创建新文件
4. 将内容粘贴到新文件中
5. 删除原文件（如果需要）

### 注意事项

- 转换后，文件的版本历史不会保留
- 建议在转换前先确认目标类型，避免重复操作
- 文件夹名称会影响新文件的默认类型：
  - "大纲"文件夹 → 大纲类型
  - "正文"或"手稿"文件夹 → 手稿类型
  - "角色"文件夹 → 角色卡类型
  - "设定"或"世界观"文件夹 → 世界观类型

## 最佳实践

### 文件组织结构

建议采用以下文件组织结构：

```
我的小说项目
├── 大纲/
│   ├── 整体架构
│   ├── 第一卷大纲
│   └── 第二卷大纲
├── 正文/
│   ├── 第一章：初入异世
│   ├── 第二章：初遇
│   └── 第三章：冲突
├── 角色/
│   ├── 主角-李青
│   ├── 女主-林婉儿
│   └── 反派-黑煞
└── 设定/
    ├── 世界结构
    ├── 修炼体系
    ├── 势力分布
    └── 历史背景
```

### 与 AI 协作的技巧

1. **明确引用**：在对话中明确告诉 AI 要参考哪些角色卡和世界观设定
2. **按需选择**：不需要每次都引用所有设定，选择相关的即可
3. **逐步完善**：AI 的建议可能不完全符合预期，可以在其基础上进行修改和完善
4. **保持更新**：及时更新角色和世界设定，让 AI 的输出与故事发展保持同步

通过合理使用不同类型的文件，您可以让zenstory 更好地理解您的创作意图，提供更精准的辅助，让写作变得更加高效和愉快。
