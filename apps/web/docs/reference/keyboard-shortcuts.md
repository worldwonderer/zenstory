# 快捷键参考

zenstory提供了丰富的键盘快捷键，帮助您更高效地进行小说创作。熟练掌握这些快捷键，能够显著提升您的写作效率。

![快捷键演示 - 展示用户使用 Cmd+K 打开文件搜索、在编辑器中使用 Cmd+S 保存文件的场景](/docs-images/reference/keyboard-shortcuts.png)

## 全局快捷键

全局快捷键在应用的任何位置都可以使用，让您快速访问常用功能。

| 快捷键 | Mac | Windows/Linux | 功能说明 |
|--------|-----|---------------|----------|
| Ctrl+K | ⌘K | Ctrl+K | **快速搜索文件** - 在当前项目中搜索文件，支持模糊匹配和键盘导航 |
| Escape | Esc | Esc | **取消/关闭** - 关闭弹窗、搜索框、取消当前操作 |

**使用技巧**：
- 使用 `Ctrl+K`（Mac: `⌘K`）打开搜索后，可用上下箭头键导航，按 Enter 打开文件
- 搜索支持文件标题的模糊匹配，无需输入完整文件名

## 编辑器快捷键

编辑器快捷键让您在写作时双手不离键盘，专注于内容创作。

| 快捷键 | Mac | Windows/Linux | 功能说明 |
|--------|-----|---------------|----------|
| Ctrl+S | ⌘S | Ctrl+S | **保存文件** - 立即保存当前文件内容，并创建版本快照 |
| Ctrl+Shift+Q | ⌘⇧Q | Ctrl+Shift+Q | **添加到引用** - 将选中的文本添加到聊天面板的引用列表 |
| Ctrl+Z | ⌘Z | Ctrl+Z | **撤销** - 撤销上一步编辑操作 |
| Ctrl+Y | ⌘⇧Z | Ctrl+Y | **重做** - 重做已撤销的操作 |
| Ctrl+Shift+Z | ⌘⇧Z | Ctrl+Shift+Z | **重做** - 重做已撤销的操作（另一种快捷键） |

**自动保存**：编辑器会在您停止输入 3 秒后自动保存内容，无需频繁手动保存。但使用 `Ctrl+S` 可以立即保存并创建版本记录。

**引用功能**：选中编辑器中的任意文本，按 `Ctrl+Shift+Q` 可将其添加到聊天引用列表，方便与 AI 讨论特定内容。

### Diff 审阅模式快捷键

当 AI 助手对文件进行修改后，编辑器会进入 Diff 审阅模式，让您可以逐个接受或拒绝修改。

| 快捷键 | Mac | Windows/Linux | 功能说明 |
|--------|-----|---------------|----------|
| Shift+Y | ⇧Y | Shift+Y | **接受所有修改** - 一键接受所有 AI 提出的修改建议 |
| Shift+N | ⇧N | Shift+N | **拒绝所有修改** - 一键拒绝所有 AI 提出的修改建议 |
| Enter | Return | Enter | **完成审阅** - 确认审阅结果，退出 Diff 模式 |
| Escape | Esc | Esc | **取消审阅** - 拒绝所有修改并退出审阅模式 |

**审阅流程**：在 Diff 模式下，您可以逐个查看修改，点击"接受"或"拒绝"按钮，或使用快捷键批量操作。

## 聊天面板快捷键

聊天面板的快捷键设计符合主流聊天软件的习惯，让您与 AI 的对话更加流畅。

| 快捷键 | Mac | Windows/Linux | 功能说明 |
|--------|-----|---------------|----------|
| Enter | Return | Enter | **发送消息** - 将当前输入的消息发送给 AI 助手 |
| Shift+Enter | ⇧Return | Shift+Enter | **换行** - 在消息中插入换行符，输入多行内容 |
| Tab | Tab | Tab | **采纳建议** - 当输入框为空时，按 Tab 键采纳第一条智能建议 |
| / | / | / | **触发技能** - 输入 "/" 打开技能快速选择菜单 |
| Escape | Esc | Esc | **关闭菜单** - 关闭技能选择菜单或清除输入 |

**智能建议**：当输入框为空时，系统会显示智能建议气泡。按 `Tab` 键可快速采纳第一条建议，或点击建议气泡选择其他建议。

**技能触发**：输入 "/" 会打开技能菜单，支持键盘导航：
- 上下箭头键：在技能列表中移动
- Enter/Tab：选择当前高亮的技能
- Escape：关闭菜单

## 文件树快捷键

文件树支持基本的键盘操作，让您无需鼠标即可管理项目文件。

| 快捷键 | Mac | Windows/Linux | 功能说明 |
|--------|-----|---------------|----------|
| F2 | F2 | F2 | **重命名** - 重命名选中的文件或文件夹（计划中） |
| Delete | Delete | Delete | **删除** - 删除选中的文件（会有确认提示） |
| Escape | Esc | Esc | **取消操作** - 取消正在进行的创建或重命名操作 |

**注意事项**：
- 删除操作会弹出确认对话框，防止误删
- 文件夹删除会同时删除其中的所有文件，请谨慎操作

## Mac 用户特别说明

如果您使用 Mac 系统，请注意以下按键对应关系：

- **⌘ (Command)** = Windows/Linux 的 Ctrl
- **⇧ (Shift)** = Shift
- **Return** = Enter

**示例**：
- Windows/Linux: `Ctrl+K` → Mac: `⌘K`
- Windows/Linux: `Ctrl+Shift+Q` → Mac: `⌘⇧Q`

## 自定义快捷键

目前zenstory暂不支持用户自定义快捷键。所有快捷键都是固定的，以确保一致的用户体验。

**计划中的功能**：
- 快捷键自定义界面
- 快捷键冲突检测
- 预设方案切换（如 Vim 模式、Emacs 模式）

如果您有特定的快捷键需求或建议，欢迎通过以下方式反馈：
- 在 GitHub 上提交 Issue
- 加入用户社区讨论群

## 快捷键记忆技巧

为了帮助您快速记住这些快捷键，这里提供一些记忆技巧：

**全局操作类**：
- **K** = **K**搜索 (Search) - `Ctrl+K` 快速搜索文件
- **S** = **S**ave (保存) - `Ctrl+S` 保存文件
- **Q** = **Q**uote (引用) - `Ctrl+Shift+Q` 添加引用

**聊天交互类**：
- **Enter** = 直接发送，符合聊天软件习惯
- **Shift+Enter** = 强制换行，需要组合键
- **/** = 技能路径，类似命令行的感觉

**审阅操作类**：
- **Y** = **Y**es (是) - `Shift+Y` 接受所有修改
- **N** = **N**o (否) - `Shift+N` 拒绝所有修改

## 常见问题

### 快捷键不生效怎么办？

1. **检查输入法**：某些中文输入法可能会拦截快捷键，尝试切换到英文输入法
2. **检查焦点**：确保焦点在正确的面板上（编辑器、聊天等）
3. **浏览器冲突**：某些浏览器快捷键可能与应用冲突，尝试禁用浏览器扩展

### 如何查看当前可用的快捷键？

目前应用内暂无快捷键帮助面板。您可以：
- 参考本文档
- 查看按钮和菜单项的 tooltip 提示
- 查看应用底部的状态栏提示

### 移动端有快捷键吗？

移动端（手机、平板）主要使用触摸操作，不支持键盘快捷键。如果您使用平板搭配外接键盘，部分快捷键可能可用，但体验可能不如桌面端完善。
