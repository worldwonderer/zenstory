# 创建第一个项目

欢迎使用zenstory小说写作工作台！这篇教程将手把手教你创建第一个创作项目，开启你的AI辅助创作之旅。

---

## 什么是项目？

在zenstory中，**项目是创作的基本单位**。一个项目就是一部完整的作品，无论是长篇小说、短篇故事还是短剧剧本，都需要先创建一个项目来承载。

项目创建后，系统会自动为你生成匹配的文件结构，包括大纲、角色、设定、正文等文件夹，让你专注于创作本身。

---

## 创建新项目

### 方法一：从首页创建

[截图: 首页Hero区域，突出显示「开始创作」按钮]

如果你还没有登录，访问zenstory官网时会看到首页：

1. 点击页面中央的「开始创作」按钮
2. 如果未登录，会跳转到登录/注册页面
3. 登录成功后自动进入仪表盘，即可创建项目

### 方法二：从仪表盘创建

[截图: 仪表盘首页，突出显示项目类型Tab和灵感输入框]

登录后进入仪表盘，你会看到一个醒目的创作区域：

1. **选择项目类型**：顶部有三个Tab（长篇小说、短篇故事、短剧剧本），点击选择你想要的类型
2. **输入创作灵感**：在文本框中描述你的故事创意
3. **点击「开始创作」**：系统会根据你的灵感创建项目

> 提示：灵感输入框支持按 Enter 键快速创建，无需点击按钮。

---

## 项目类型选择

[截图: 项目类型选择器，展示三个类型的图标和说明]

zenstory提供三种项目类型，每种都有预设的文件结构：

| 类型 | 字数范围 | 适用场景 | 预设文件夹 |
|------|----------|----------|------------|
| **长篇小说** | 5万字以上 | 连载小说、网文、长篇故事 | 设定、角色、素材、大纲、正文 |
| **短篇故事** | 5000-20000字 | 独立短篇、参赛作品 | 人物、构思、正文 |
| **短剧剧本** | 不限 | 短视频剧本、微短剧 | 角色、场景、分集大纲、剧本 |

**如何选择？**

- 如果你在写一部需要多章节、多角色的大型作品，选择「长篇小说」
- 如果你想写一个完整但篇幅有限的独立故事，选择「短篇故事」
- 如果你在创作适合短视频平台的剧本，选择「短剧剧本」

> 建议：选择最符合你创作规模的类型，系统会提供最合适的文件夹结构。类型一旦选定，后续可以通过手动调整文件结构来适应需求变化。

## 填写初始灵感

[截图: 灵感输入框，展示示例内容]

初始灵感是项目的起点。AI会根据你提供的灵感，自动生成初始框架，帮你快速开始创作。

### 什么是好的灵感描述？

一个好的灵感描述应该包含：

- **核心创意**：故事最独特的点是什么？
- **主要角色**：主角是谁？有什么特点？
- **故事背景**：发生在什么世界？什么时代？
- **核心冲突**：主角面临什么挑战或困境？

### 示例输入

**长篇小说示例：**

```
我想写一部都市异能小说，主角是一个普通大学生，某天意外获得了时间回溯的能力，
每次死亡都会回到24小时前。故事围绕他如何在一次次死亡中成长，最终揭开城市背后
的神秘组织展开。风格偏悬疑，有少量爱情线。
```

**短篇故事示例：**

```
一个老人每天都会去公园的长椅上坐一整天，风雨无阻。直到有一天，一个小孩递给他
一颗糖，揭开了他心中尘封多年的秘密。温情治愈向，探讨记忆与陪伴的主题。
```

**短剧剧本示例：**

```
甜宠职场短剧：新来的实习生竟然是公司总裁的女儿，她隐瞒身份入职想要证明自己，
却意外爱上了同样隐瞒身份的集团继承人。每集2分钟，主打轻喜剧和甜蜜日常。
```

### 不确定写什么？

如果暂时没有明确的灵感，你也可以：

1. 输入一个简单的主题（如"古代修仙"、"现代都市"）
2. 创建项目后，与AI助手对话，让它帮你头脑风暴
3. 先创建角色和世界观设定，再逐步发展故事

## 项目创建后的初始文件

[截图: 项目编辑页面，展示左侧文件树]

项目创建成功后，系统会自动跳转到项目编辑页面。左侧的文件树会显示预设的文件夹结构。

### 自动创建的内容

根据项目类型，你会看到以下文件夹：

**长篇小说：**
- 📁 **设定** - 世界观、魔法体系、背景设定
- 📁 **角色** - 角色档案、人物关系
- 📁 **素材** - 灵感片段、参考资料
- 📁 **大纲** - 章节大纲、剧情规划
- 📁 **正文** - 小说正文内容

**短篇故事：**
- 📁 **人物** - 角色设定
- 📁 **构思** - 故事框架
- 📁 **正文** - 故事正文

**短剧剧本：**
- 📁 **角色** - 人物设定
- 📁 **场景** - 场景描述
- 📁 **分集大纲** - 每集剧情
- 📁 **剧本** - 正式剧本

### 如果提供了初始灵感

如果你在创建项目时填写了初始灵感，AI会自动开始工作：

1. 页面右侧的AI助手面板会显示「正在思考...」
2. AI会根据你的灵感生成大纲框架
3. 可能还会创建初始角色卡或世界观设定
4. 生成的内容会自动保存到对应的文件中

你可以在聊天面板中看到AI的完整思考过程和执行步骤。

## 下一步：开始创作

项目创建完成后，你就可以正式开始创作了！

### 推荐阅读

- **[AI创作助手使用指南](../user-guide/ai-assistant.md)** - 学习如何与AI对话，让它帮你写作
- **[文件管理指南](../user-guide/file-tree.md)** - 了解如何管理你的创作文件
- **[界面概览](../user-guide/interface-overview.md)** - 熟悉zenstory的三栏式工作界面

### 常见创作流程

1. **完善设定** - 先建立世界观、魔法体系等背景
2. **设计角色** - 创建主要角色的详细档案
3. **规划大纲** - 让AI帮你生成章节大纲
4. **撰写正文** - 逐章写作，AI可以续写、扩写、改写
5. **导出作品** - 完成后导出为 `.txt` 文件

### 快速上手建议

如果你是第一次使用zenstory，建议这样做：

1. 在右侧聊天面板输入：「帮我完善一下这个故事的开头设定」
2. AI会根据你的初始灵感，生成更详细的世界观和角色
3. 查看生成的文件，觉得不满意可以要求AI修改
4. 然后输入：「根据现在的设定，帮我写第一章的大纲」
5. 大纲确认后，输入：「根据第一章大纲，帮我写第一章的正文」

就这样，一步一个脚印，AI会成为你最可靠的创作伙伴！
