# 5分钟快速入门

欢迎使用 zenstory AI 小说写作工作台！这份教程将帮助你在 5 分钟内快速上手，开始你的 AI 辅助创作之旅。

## 你将学会

- 注册与登录账户
- 创建你的第一个创作项目
- 与AI助手对话进行创作
- 管理创作文件
- 导出你的作品

---

## 第一步：注册与登录

### 方式一：邮箱注册（推荐）

1. 访问zenstory官网，点击右上角「注册」按钮
2. 填写用户名、邮箱地址和密码
3. 如果你有邀请码，可以在注册时填写获得额外奖励（100积分，完成邮箱验证后发放）
4. 点击「注册」后，系统会向你的邮箱发送验证码
5. 输入收到的6位验证码完成验证
6. 验证成功后自动登录，进入工作台

### 方式二：Google OAuth快捷登录

1. 在登录页面点击「使用Google登录」按钮
2. 在弹出的Google授权页面选择你的Google账户
3. 授权后自动完成注册/登录，无需额外验证

> 提示：使用Google登录更便捷，无需记忆额外密码，也省去了邮箱验证步骤。

### 关于邀请码系统

zenstory采用邀请制注册，新用户需要邀请码才能注册：

- **如何获取邀请码**：向已注册的用户索取，或在官方社区申请
- **使用邀请码的奖励**：
  - 邀请人获得100积分
  - 被邀请人获得100积分
- **防刷机制**：系统会检测异常注册行为，确保公平使用

> 详细说明请参阅 [账号注册与登录](./installation.md)。

## 第二步：创建你的第一个项目

### 2.1 开始创建

1. 登录后进入Dashboard首页
2. 点击页面中央的「新建项目」按钮（或右上角的「+」图标）
3. 弹出项目创建对话框

### 2.2 选择项目类型

zenstory支持三种创作类型，根据你的需求选择：

| 项目类型 | 适合场景 | 预设文件夹 |
|---------|---------|-----------|
| **长篇小说** | 5万字以上的连载小说 | 设定、角色、素材、大纲、正文 |
| **短篇小说** | 5000-20000字的独立短篇 | 人物、构思、正文 |
| **短剧剧本** | 短视频剧本、微短剧 | 角色、场景、分集大纲、剧本 |

> 建议：根据创作规模选择合适类型，系统会自动创建对应的文件夹结构。

### 2.3 填写项目信息

1. **项目名称**：给你的作品起个名字（如「我的第一本科幻小说」）
2. **初始灵感**（可选）：描述你的创作想法，AI会据此生成初始内容

**示例初始灵感**：

```
我想写一部都市异能小说，主角是一个普通大学生，
某天意外获得了时间回溯的能力，每次死亡都会回到24小时前。
```

3. 点击「创建项目」

### 2.4 开始创作

项目创建成功后，系统会：

- 自动创建预设文件夹（大纲、角色等）
- 如果你填写了初始灵感，AI会自动开始创作，生成大纲和第一章草稿

> 详细说明请参阅 [创建第一个项目](./first-project.md)。

## 第三步：与AI助手对话

### 3.1 认识工作界面

zenstory采用三栏式布局：

- **左侧：文件树** - 管理所有创作文件
- **中间：编辑器** - 编辑文件内容
- **右侧：AI助手** - 与AI对话，获取创作帮助

### 3.2 如何与AI对话

1. 在右侧聊天面板底部的输入框中输入你的需求
2. 点击发送按钮（或按Enter键）
3. AI会思考你的需求，然后执行相应操作

### 3.3 常用对话示例

**创建角色**：

```
帮我创建一个女主角，名叫林晓月，是大学中文系学生，
性格开朗活泼，喜欢古典文学。
```

**生成大纲**：

```
根据我的设定，帮我生成一个10章的小说大纲，
每章包含200-300字的剧情梗概。
```

**撰写正文**：

```
根据第一章大纲，帮我写第一章的正文，大约3000字。
```

**修改内容**：

```
把第三章中主角的对话改得更幽默一些。
```

**请求建议**：

```
我觉得第三章的剧情冲突不够，你有什么建议吗？
```

### 3.4 理解AI的思考过程

当你发送消息后，AI会：

1. **思考中**（显示「AI正在思考...」）- AI分析你的需求，规划执行步骤
2. **执行工具** - AI调用相应工具（创建文件、修改内容等）
3. **展示结果** - 显示操作结果和AI的回复

你可以在聊天面板中看到AI的完整思考和执行过程，确保创作透明可控。

### 3.5 快捷操作提示

- **附加文件**：在发送消息前，可以附加相关文件作为上下文
- **引用文本**：选中编辑器中的文本，右键选择「引用到对话」，让AI针对具体内容修改
- **语音输入**：点击输入框右侧的麦克风按钮，语音转文字输入（需浏览器支持）
- **智能建议**：AI会在对话结束后提供后续创作建议，点击即可快速发送

> 详细说明请参阅 [AI创作助手](../user-guide/ai-assistant.md)。

## 第四步：管理你的文件

### 4.1 文件类型说明

zenstory支持多种文件类型：

| 文件类型 | 用途 |
|---------|------|
| **设定** | 世界观、魔法体系、背景设定 |
| **角色** | 角色档案、人物关系 |
| **大纲** | 章节大纲、剧情规划 |
| **正文** | 小说正文、实际内容 |
| **素材** | 灵感片段、参考资料 |

### 4.2 文件操作

**新建文件**：

1. 将鼠标悬停在文件夹上，点击「+」按钮
2. 输入文件名称
3. 选择文件类型（系统会根据所在文件夹自动推荐）
4. 点击确认

**编辑文件**：

1. 在文件树中点击文件名
2. 文件内容会在中间编辑器中打开
3. 编辑内容会自动保存

**删除文件**：

1. 将鼠标悬停在文件上，点击垃圾桶图标
2. 确认删除操作
3. 删除的文件可以在版本历史中恢复

**移动文件**：

1. 拖拽文件到目标文件夹
2. 松开鼠标完成移动

### 4.3 文件搜索

快速查找文件：

- **快捷键**：Mac按 `Cmd + K`，Windows按 `Ctrl + K`
- 输入文件名关键词进行模糊搜索
- 支持按文件类型筛选（大纲、正文、角色等）
- 使用键盘上下键选择，Enter键打开文件

### 4.4 版本历史

zenstory会自动保存每次修改的版本：

1. 打开任意文件
2. 点击编辑器顶部的「版本历史」按钮
3. 查看所有历史版本
4. 选择任意版本进行对比或回滚

> 提示：AI每次修改文件后都会自动创建快照，你可以随时回退到之前的版本。

> 详细说明请参阅 [文件树与文件类型](../user-guide/file-tree.md) 和 [版本历史](../user-guide/version-history.md)。

## 第五步：导出作品

完成创作后，可以将作品导出为本地文件。

### 5.1 导出步骤

1. 在项目页面，点击顶部工具栏的「导出」按钮
2. 点击「开始导出」
3. 浏览器会自动下载导出的文件（`.txt`）

### 5.2 导出内容说明

- 导出范围：当前项目的所有正文文件
- 排序方式：按文件创建顺序排列
- 文件命名：`{项目名称}_正文.txt`

### 5.3 导出后的使用

导出的文件可以：

- 在Word、WPS等文字处理软件中打开编辑
- 上传到小说平台进行发布
- 分享给朋友或编辑审阅
- 打印成纸质稿

> 详细说明请参阅 [导出功能](../user-guide/export.md)。
