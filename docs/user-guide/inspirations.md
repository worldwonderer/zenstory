# 灵感库

灵感库用于沉淀高质量创意模板，帮助你更快从「想法」进入「可写内容」。

## 入口位置

- 仪表盘侧边栏：**灵感库**
- 仪表盘首页：**精选灵感** 卡片点击「查看全部」

## 核心能力

### 1) 浏览与筛选

- 按分类/标签筛选灵感
- 通过关键词搜索灵感标题与摘要
- 查看推荐与最新内容

### 2) 查看灵感详情

灵感详情页通常包含：

- 标题与一句话卖点
- 题材/标签
- 背景设定与角色关系
- 冲突核心与可展开方向

### 3) 复制到项目

在灵感详情页可直接将内容复制到当前项目，常见落位：

- 大纲文件（剧情骨架）
- 角色文件（人物原型）
- 设定文件（世界观与规则）

复制后建议你先做一次本地化改写，避免模板感过强。

## 推荐使用流程

1. 在灵感库选 1 条最贴近当前创作目标的灵感  
2. 复制到项目的大纲/角色/设定文件  
3. 在右侧 AI 助手中明确指令：  
   - 「基于该灵感，给我 3 个不同走向」
   - 「保留核心冲突，改成悬疑基调」
4. 选定方向后再展开章节

## 常见问题

### 灵感库和素材库有什么区别？

- **灵感库**：偏「创意模板与方向」  
- **素材库**：偏「参考小说拆解后的结构化资料」

两者可以配合使用：先用灵感库定方向，再用素材库补细节。
