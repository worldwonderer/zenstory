# 技能系统

技能系统是zenstory 写作工作台的核心功能，通过预设的指令模板快速调用 AI 完成写作任务。只需在对话框中输入触发词，AI 就会自动应用对应的技能指令。

## 什么是技能？

技能是预设的 AI 指令模板，通过触发词激活。当你在对话中输入包含触发词的内容时，系统会自动识别并应用对应的指令，让 AI 按照预设方式响应。

**核心优势**：
- 快速调用复杂指令，无需重复输入
- 确保 AI 输出风格的一致性
- 可根据个人习惯定制专属技能
- 发现和使用社区分享的优质技能

## 发现技能

[截图: 发现技能页面，展示技能卡片网格、分类筛选和搜索框]

在「发现技能」页面浏览和添加来自官方和社区的优质技能：

**功能操作**：
- **搜索**：输入关键词快速找到所需技能
- **分类筛选**：点击分类标签（写作技巧、角色塑造等）筛选
- **添加技能**：点击「添加到我的技能」将技能加入个人库
- **展开查看**：点击展开按钮查看完整指令内容

**技能来源**：
| 来源 | 说明 |
|------|------|
| 官方 | zenstory团队精心打造，经过充分测试优化 |
| 社区 | 用户创建并分享，经审核后发布 |

## 我的技能

[截图: 我的技能页面，分为「自定义技能」和「已添加技能」两个区域]

管理所有个人技能，分为两类：

| 类型 | 说明 | 权限 |
|------|------|------|
| 自定义技能 | 亲手创建的技能 | 可编辑、删除、分享 |
| 已添加技能 | 从公共库添加的技能 | 只读，可移除 |

**管理功能**：
- 搜索：按名称或描述快速定位技能
- 批量操作：选择多个技能后批量删除

## 创建自定义技能

[截图: 创建技能表单，包含名称、描述、触发词、指令内容四个输入字段]

### 基本信息

- **技能名称**（必填）：简洁明了的名字
- **描述**（选填）：说明用途和使用场景

### 触发词设置

[截图: 触发词输入框]

触发词是激活技能的关键。设置要点：
- 多个触发词用英文逗号分隔
- 使用自然语言短语，便于记忆
- 避免过于常见的词汇

**示例**：`创建角色, 新建人物, 角色卡`

### 指令内容

[截图: 指令编辑器]

AI 将遵循的详细指令，支持 Markdown 格式。

**编写建议**：
- 明确 AI 的角色和任务
- 提供输出格式要求
- 使用 Markdown 结构化内容

**示例指令**：

```markdown
# 角色创建专家

你是一位专业的小说角色设计顾问。请按以下格式生成角色设定：

## 基础信息
- 姓名、年龄、性别、身份

## 外貌特征
详细描述角色的外貌特点。

## 性格特点
- 核心性格、行为习惯、语言风格

## 背景故事
简要描述成长经历和重要事件。

请确保角色设定与故事风格保持一致。
```

## 编辑与删除技能

**编辑**：点击自定义技能卡片上的编辑图标修改内容。已添加的公共技能为只读，无法编辑。

**删除**：点击删除图标，确认后永久删除。支持批量删除多个技能。

**注意**：删除操作无法撤销，请谨慎操作。

## 分享技能

[截图: 分享技能弹窗，包含分类选择]

创建有价值的技能后可分享到公共库：

1. 点击分享图标
2. 选择分类（写作技巧、角色塑造、世界观构建、情节设计、文风润色）
3. 提交分享

分享的技能会提交管理员审核，通过后出现在公共库中。

## 技能使用统计

[截图: 技能统计弹窗]

点击「使用统计」查看当前项目的技能使用情况：

**统计指标**：
- 总触发次数、平均置信度
- 内置技能 vs 自定义技能触发次数

**时间范围**：7天 / 30天 / 90天

**可视化**：
- 最常用技能排行（Top 5）
- 每日使用趋势柱状图

## 使用技巧

**触发词设计**：
- 选择自然语言短语
- 避免不同技能触发词冲突
- 每个技能 2-5 个触发词为宜

**指令编写**：
- 使用 Markdown 结构化组织
- 包含期望的输出示例
- 根据 AI 输出持续优化

**技能管理**：
- 定期清理不再使用的技能
- 使用统一的命名规范
- 有价值的技能及时分享
