# 术语表

本文档列出了zenstory 小说写作工作台中使用的所有专业术语，帮助您更好地理解产品功能和相关概念。

## 核心概念

| 术语 | 英文 | 解释 |
|------|------|------|
| 项目 | Project | 一部作品的容器，包含所有相关的大纲、手稿、角色卡、世界观等文件 |
| 文件树 | File Tree | 项目中文件的层级结构展示，支持文件夹嵌套组织 |
| 大纲 | Outline | 故事的章节结构和情节规划文件 |
| 手稿 | Draft | 正文内容文件，您的实际创作内容 |
| 角色卡 | Character | 角色设定的文档，记录角色的外貌、性格、背景等信息 |
| 世界观 | Lore | 世界设定的文档，包括力量体系、势力分布、特殊规则等 |
| 素材 | Snippet | 文本素材文件，用于存储灵感片段、参考资料等 |
| 剧本 | Script | 短剧专用的剧本内容文件 |

## AI 相关

| 术语 | 英文 | 解释 |
|------|------|------|
| AI 助手 | AI Assistant | 您的智能写作伙伴，可以帮助生成内容、优化文字、构建设定等 |
| 技能 | Skill | 预设的 AI 指令模板，通过触发词快速调用 |
| 触发词 | Trigger Word | 激活技能的关键词，在对话中输入即可自动匹配对应技能 |
| AI 记忆 | AI Memory | 项目级别的持久信息，包括项目简介、写作风格、当前阶段等，所有对话都会参考 |
| 上下文 | Context | AI 对话时参考的背景信息，包括当前文件、附加素材、引用文本等 |
| 焦点文件 | Focus File | 当前在编辑器中打开的文件，AI 会自动获取其内容作为上下文 |
| 思考过程 | Thinking | AI 处理复杂任务时的推理逻辑，透明展示给用户 |
| 工具调用 | Tool Call | AI 执行的具体操作，如创建文件、编辑内容、查询信息等 |

## 版本管理

| 术语 | 英文 | 解释 |
|------|------|------|
| 版本历史 | Version History | 文件的完整修改记录，每次保存都会自动创建新版本 |
| 版本快照 | Snapshot | 某个时间点的完整文件内容记录 |
| 版本对比 | Diff | 两个版本之间的内容差异对比，新增显示绿色，删除显示红色 |
| 回滚 | Rollback | 恢复到历史版本的操作，当前内容会自动保存为新版本 |
| 自动保存 | Auto Save | 系统自动保存编辑内容，无需手动操作 |

## 素材库

| 术语 | 英文 | 解释 |
|------|------|------|
| 素材库 | Materials Library | 上传参考小说并利用 AI 智能拆解分析的功能 |
| AI 拆解 | AI Analysis | 系统自动识别并提取章节结构、角色信息、剧情线、世界观等元素 |
| 章节结构 | Chapter Structure | AI 识别的章节划分、标题、摘要和情节点 |
| 剧情线 | Storyline | 小说中的主要剧情线索，包括核心目标、冲突和情节类型 |
| 金手指 | Cheat/Power | 主角或重要角色的特殊能力、系统或优势设定 |
| 时间线 | Timeline | 小说中事件的先后顺序梳理 |

## 界面交互

| 术语 | 英文 | 解释 |
|------|------|------|
| 三栏布局 | Three-Panel Layout | 桌面端的界面结构，左侧文件树、中间编辑器、右侧 AI 助手 |
| 左侧栏 | Left Sidebar | 项目资源区，包含文件、技能、素材三个 Tab |
| 编辑器 | Editor | 中间的核心创作区域，支持富文本编辑 |
| 差异审核模式 | Review Mode | AI 编辑文件后，逐条审核每处修改的模式 |
| 引用文本 | Quote Text | 在编辑器中选中文字并附加到对话，让 AI 针对处理 |
| 附加素材 | Attach Material | 在对话中选择相关文件作为 AI 的参考上下文 |
| 建议提示 | Suggestion | AI 根据项目状态动态生成的快捷指令建议 |
| 语音输入 | Voice Input | 通过麦克风录音并转换为文字的输入方式 |
| 流式输出 | Streaming | AI 逐字生成内容并实时显示的输出方式 |

## 其他

| 术语 | 英文 | 解释 |
|------|------|------|
| 导出 | Export | 将项目正文导出为纯文本（.txt）文件（当前版本） |
| 自动备份 | Auto Backup | 系统定期自动保存所有内容到数据库 |
| 软删除 | Soft Delete | 删除的文件不会立即永久删除，可以通过特殊方式恢复 |
| 深色模式 | Dark Mode | 适合夜间创作的暗色界面主题 |
| 主题色 | Theme Color | 界面的强调色，可以个性化设置 |
| 响应式设计 | Responsive Design | 界面根据设备屏幕尺寸自动调整布局 |

掌握这些术语将帮助您更好地使用zenstory 小说写作工作台。如有任何疑问，请参考相关的详细文档或联系客服获取帮助。
