# 错误信息说明

本文档列出了使用zenstory时可能遇到的常见错误信息及其解决方法，帮助您快速定位问题并恢复使用。

---

## 常见错误信息

### 通用错误

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 服务器内部错误，请稍后重试 | 服务器遇到未预期的错误 | 刷新页面重试；如持续出现，请联系客服 |
| 请求参数验证失败 | 提交的数据格式不正确 | 检查输入内容是否符合要求（如邮箱格式、必填项等） |
| 数据库操作失败 | 数据保存或读取时出错 | 稍后重试；如持续出现，请联系客服 |
| 服务暂时不可用 | 系统维护或服务过载 | 等待几分钟后重试 |
| 请求格式错误 | API 请求格式不符合规范 | 刷新页面，清除浏览器缓存后重试 |
| 资源不存在 | 访问的内容已被删除或不存在 | 返回上一页，检查资源是否仍存在 |

### 账号与登录

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 用户名或密码错误 | 输入的账号信息不正确 | 检查邮箱和密码是否正确；尝试找回密码 |
| 登录已过期，请重新登录 | Token 已过期 | 重新登录即可恢复 |
| 无效的登录凭证 | Token 无效或被篡改 | 退出登录后重新登录 |
| 账户已被禁用 | 账号被管理员禁用 | 联系客服了解原因并申请恢复 |
| 用户名已被注册 | 该用户名已存在 | 更换用户名注册 |
| 邮箱已被注册 | 该邮箱已注册过账号 | 直接登录，或使用其他邮箱注册 |
| 邮箱未验证，请先验证邮箱 | 注册后未完成邮箱验证 | 检查邮箱（包括垃圾邮件）获取验证链接 |
| 验证码无效或已过期 | 验证码已过期或输入错误 | 重新发送验证码 |

### 项目相关

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 项目不存在 | 项目可能已被删除 | 返回项目列表，确认项目是否存在 |
| 项目已存在 | 同名项目已创建 | 更换项目名称 |
| 项目已删除 | 项目处于回收站状态 | 在「已删除」中恢复项目 |
| 项目中没有草稿可导出 | 项目内无草稿类型文件 | 先创建草稿内容后再导出 |
| 您没有权限执行此操作 | 当前账号无该项目权限 | 确认是否为项目所有者 |

### 文件相关

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 文件不存在 | 文件可能已被删除 | 刷新页面，检查文件树 |
| 文件类型无效 | 文件类型不在支持范围内 | 使用支持的文件类型（大纲、草稿、角色、设定、素材） |
| 向量搜索服务不可用 | AI 搜索功能暂时离线 | 稍后重试；功能恢复后可继续使用 |

### 版本与快照

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 版本不存在 | 指定的版本记录不存在 | 刷新版本历史列表 |
| 未找到文件的版本 | 该文件尚未创建任何版本 | 编辑并保存文件后会自动创建版本 |
| 删除版本失败 | 版本删除操作失败 | 稍后重试 |
| 恢复版本失败 | 版本恢复操作失败 | 稍后重试 |
| 快照不存在 | 指定的快照不存在 | 刷新页面后重试 |
| 比较快照失败 | 快照对比时出错 | 稍后重试 |
| 创建快照失败 | 快照创建失败 | 检查文件是否有内容，稍后重试 |

### 对话与语音

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 对话会话不存在 | 对话记录已被删除 | 创建新的对话会话 |
| 消息不存在 | 消息记录不存在 | 刷新对话列表 |
| 腾讯云 API 凭证未配置 | 语音识别服务未启用 | 该功能需要管理员配置，请联系客服 |
| 音频数据解码失败 | 录音文件格式或内容有问题 | 重新录音后提交 |
| 语音识别 API 请求失败 | 语音服务暂时不可用 | 稍后重试 |

### 邀请码

| 错误信息 | 原因 | 解决方法 |
|----------|------|----------|
| 邀请码无效 | 输入的邀请码不存在 | 检查邀请码格式（如 `A1B2-C3D4`）是否正确 |
| 邀请码已过期 | 邀请码已超过有效期 | 获取新的邀请码 |
| 邀请码已用完 | 邀请码使用次数已达上限 | 获取新的邀请码 |
| 已达到邀请码数量上限 | 每位用户最多创建 3 个邀请码 | 删除不用的邀请码后再创建 |

## 如何报告错误

如果您遇到的错误未在上述列表中，或问题持续无法解决，请按以下步骤提交反馈：

1. **记录错误信息**：复制完整的错误提示文字
2. **截图错误界面**：包含错误弹窗或高亮区域
3. **描述操作步骤**：详细说明出现错误前的操作
4. **提供设备信息**：浏览器类型、版本、操作系统

---

## 联系支持

提交问题时，请附上以下信息以加快处理速度：

- **错误截图**：包含完整错误信息
- **复现步骤**：一步步描述如何触发错误
- **设备信息**：浏览器版本（如 Chrome 120）、操作系统（如 macOS 14）
- **账号信息**：出现问题时的账号邮箱（可脱敏处理）

我们将尽快处理您反馈的问题，感谢您的支持与理解！
