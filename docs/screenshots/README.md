# Screenshots

截图尺寸建议：宽度 1200px，PNG 格式。

## 需要的截图

| 文件名 | 内容 |
|--------|------|
| `banner.png` | 项目 Logo 或 Hero 图（600x200） |
| `workspace.png` | 三栏工作台全貌（文件树 + 编辑器 + AI 对话） |
| `ai-chat.png` | AI 对话写作特写，展示生成内容的过程 |
| `file-tree.png` | 文件树管理界面，展示多类型文件组织 |
| `version-history.png` | 版本历史对比界面 |

## 如何截图

1. 启动前后端，登录后进入一个包含示例文件的项目
2. 浏览器窗口宽度设为 1440px
3. 截图后裁剪到合适区域，导出为 PNG
4. 放入此目录即可
